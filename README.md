# Краткое описание проекта
Проект представляет собой клиент–серверное приложение для ведения и анализа семейного бюджета с возможностью синхронизации данных между несколькими устройствами. Пользователь может добавлять доходы и расходы, распределять их по категориям, устанавливать лимиты и цели, анализировать финансовое состояние с помощью таблиц и графиков, визуализирующих доходы и расходы.
# Функциональность проекта
## Со стороны клиента
* регистрация и вход пользователя
* добавление доходов и расходов
* визуализация данных
## Со стороны сервера
* регистрация пользователей
* проверка логина и пароля
* хранение истории операций
* автоматический пересчет баланса
# Используемые библиотеки
Qt, Boost.Asio, nlohmann/json, SQLite
# MVP
* Вход и регистрация пользователя
* Ввод операции (доход/расход)
* Вычисление баланса на сервере
* Возможность просмотреть сделанные операции по запросу
